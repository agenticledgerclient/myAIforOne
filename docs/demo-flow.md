## 1. Live Agent Creation

**Feature:** Agent Builder / @agentcreator

Start the demo by creating a new agent live — name it for the prospect's company or use case. Let it build in the background while you walk through the rest of the platform.

**What to show:**
- Open the Web UI
- Message @agentcreator: "Create an agent for [prospect's use case]"
- Watch it start building — then move on

**Key message:** "While this builds itself, let me show you what agents already do."

---

## 2. Specialized Agent Walkthrough

**Feature:** Workspaces, System Prompts, Tool Permissions

Open an existing client setup and walk through two purpose-built agents — each with a different workspace, system prompt, and tool set.

**What to show:**
- Agent detail panel — workspace path, system prompt, allowed tools
- Two agents with completely different specializations
- How each agent "knows" its domain

**Key message:** "Same platform, completely different capabilities. Each agent is an expert in its lane."

## 3. App-Building Agent

**Feature:** Agentic App Delivery + Embedded Chatbot

Show a live website or app that was built by an agent. Demo the chatbot embedded on the site.

**What to show:**
- The live site (e.g., Finance Is Cooked or similar)
- The chatbot answering domain-specific questions
- The agent that built and maintains it

**Key message:** "Agents don't just answer questions — they build and ship real products."

## 4. Action Agents

**What to show:**

**Key message:** "This isn't a chatbot. It takes action in the real world — you just tell it what to do."

## 5. Autonomous Agents

## 14. Engagement Models — The Close

**Feature:** How We Work Together

Present the four ways to engage:

1. **Training** — We teach your teams to build and use agents themselves
2. **Deployment** — We install the platform for your teams to use
3. **Agentic Apps** — We build and deliver custom agent-powered tools for your specific needs
4. **Custom Platform** — We rebuild a branded, customized version of the platform for your organization

**Key message:** "However you want to start, we meet you there. Train, deploy, build, or customize."

---

## Demo Tips

- **Start with creation** — building an agent live is the strongest opener
- **Let it cook in the background** — creates a natural "reveal" moment at Step 10
- **Match examples to the prospect** — swap in agents relevant to their industry
- **End with engagement models** — always close with a clear next step
- **Total time:** ~30 minutes for full flow, ~15 minutes for highlights only
