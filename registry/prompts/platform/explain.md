Explain the following concept as if to someone encountering it for the first time:

- Start with a real-world analogy
- Build up from first principles — no assumed knowledge
- Avoid jargon; if a technical term is unavoidable, define it inline
- Use short sentences and concrete examples
- End with a one-line summary: "In short: ..."
