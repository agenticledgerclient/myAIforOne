After installing — how do you assign to an agent?

The card changes state after install. How should the assign step work?

Inline on the card — dropdown appears after install

Card shows an "Assign to agent" dropdown right on the card. Fast, no modal. Optional — you can skip it and assign later from the agent config.

Install only — assign from agent config separately

Marketplace just handles install. Card flips to "Installed" state. To assign, go to the agent in Org → edit → pick from available MCPs/skills list. Clean separation, two completely separate flows.
